\chapter{Data and code availability}

The LaTeX source code of this thesis, along with the pipeline configuration,
the most important pipeline results and the postprocessing scripts, are
available on GitHub at \url{https://github.com/nictru/bsc-thesis}.

The exact version of the \gls{nf-circrna} pipeline used in this thesis can be
found at
\url{https://github.com/nf-core/circrna/tree/30daac450cba31a01fa5e7b2d98b834ceef9d9c9}.

The data used in this thesis can be downloaded from GEO:
\begin{itemize}
    \item Anti-hormonal dataset: GSE201326
    \item Aging dataset: GSE201767
\end{itemize}
